# Billing Management System (Java)

This project is a simple **Billing Management System** developed using **Java**.  
It demonstrates the use of important Object-Oriented Programming concepts such as **Collections, Generics, and Exception Handling**.

The application allows users to enter product details like **name, price, and quantity**, and automatically calculates the final bill amount.

---

## Features

- Add multiple products
- Automatic bill calculation
- Uses **ArrayList Collection**
- Demonstrates **Generics**
- Implements **Exception Handling**
- Simple **console-based interface**

---

## Technologies Used

- Java
- OOP Concepts
- Java Collections Framework
- Exception Handling
